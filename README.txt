Installation and Usage Instructions:

1. Make sure Minecraft is installed in the default directory:
   → %appdata%\.minecraft

2. Right-click the .exe file and select "Run as administrator".

3. An active internet connection is REQUIRED to launch the cheat.
   (The cheat needs to connect to the authentication server)

4. Allow administrator privileges when prompted by Windows.

5. Launch the .exe BEFORE starting Minecraft.
   The cheat will automatically inject into the game.

================================================================================
                          IMPORTANT NOTES
================================================================================

• Temporarily disable your antivirus software (including Windows Defender)
  as it may flag or block the cheat.

• Do not launch Minecraft before running the cheat .exe.

• Ensure you are using a compatible version of Minecraft (1.21.+ , 1.20.+) .

• The cheat requires an internet connection on every launch for activation.

• For best results, close unnecessary programs before injecting.

================================================================================
Troubleshooting

If the cheat does not work:
- Confirm you ran the .exe as Administrator
- Verify your internet connection is stable
- Make sure the .minecraft folder exists in %appdata%
- Check that your Minecraft version is supported
